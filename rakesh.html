Flight Ticket Price Prediction

Machine learning system using Python and Random Forest to predict flight prices. Includes data preprocessing, model evaluation, and Tkinter GUI.

The Flight Ticket Price Prediction project is a machine learning–based application designed to estimate airline ticket prices using historical flight data. The goal of this project is to help users understand price trends and make better booking decisions. In this project, I performed data preprocessing, feature engineering, and model training using Python. A Random Forest algorithm was implemented to achieve accurate price predictions. The trained model was evaluated for performance and accuracy using suitable metrics. To make the system user-friendly, a Tkinter-based graphical user interface (GUI) was developed, allowing users to enter flight details and instantly receive predicted ticket prices. This project strengthened my understanding of machine learning workflows, data analysis, and real-world application development.
